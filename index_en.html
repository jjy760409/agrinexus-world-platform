🌿 AgriNexus World - Unified Smart Agriculture Platform

This is the official portal integrating contract generation, AI report tools, seed recommendation, smart farming layout generator, real-time chart analytics, and Firebase-based log and data storage.

✅ Core Features

  • 📄 Contract Generator: Fill and download as PDF
  • 📑 Report Generator: Type summary → Export to PDF
  • 🤖 OpenAI Prompt Generator: Instant GPT responses
  • 🌱 SmartFarm Designer: Layout planning by area/crop
  • 🌾 Seed Recommender: Auto match seeds per country/season
  • 📊 Chart.js Price Graph: Real-time data visualization
  • 📥 Firebase Logging: Save and view logs/admin data
  • 🧠 Global Policy Portal: Worldwide fund/support insights
