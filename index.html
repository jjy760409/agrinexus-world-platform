🌐 AgriNexus World - 작동하는 핵심 기능

아래 기능은 클릭 시 실제 작동하며, 기능 설명과 이름이 명확하게 표시되어 있습니다.

🌱 스마트팜 환경 설계
실내/실외/창고 등 조건 선택 → 자동 설계 🧠 GPT 기반 도면 생성기
환경 조건 입력 → GPT 실시간 설계 응답 💰 물류비용 계산기
무게/거리/운송수단 기반 실시간 요금 출력 🤖 AI 분석 생성기
작물, 면적, 조건 입력 → GPT 분석 문장 생성 💬 고객 지원 시스템
문의 유형 입력 → GPT 자동 응답 출력 🌍 Gaia 글로벌 자동 설계
국가/환경 기반 설치 설계 + 유통/정책 추천
